# BCI Dataset Organization

This directory contains all EEG datasets used for development, testing, and validation.

## Directory Structure

```
data/
├── raw/                    # Original, unmodified datasets
│   └── sample_eeg_data.csv # Original simulated EEG data (2562 samples)
│
├── test_cases/             # Focused test datasets for unit testing
│   ├── single_channel/     # Single-channel test data
│   └── multi_channel/      # Multi-channel test data (8-16 channels)
│
├── validation/             # Integration test datasets
│   └── (full scenario test data)
│
└── synthetic/              # Programmatically generated test signals
    └── (generated by test scripts)
```

## Data Format

### CSV Format
All CSV files follow this format:
```csv
time,amplitude,alpha_power,beta_power,command
0.000,3.07,0.5,0.5,NONE
0.004,20.99,0.5,0.5,NONE
...
```

**Columns:**
- `time` (float): Timestamp in seconds
- `amplitude` (float): Signal amplitude in microvolts
- `alpha_power` (float): Alpha band power (8-13 Hz) - normalized 0-1
- `beta_power` (float): Beta band power (13-30 Hz) - normalized 0-1
- `command` (string): Mental command (NONE, FOCUS, RELAX, BLINK)

### Multi-Channel Format (future)
Multi-channel CSV will include columns for each channel:
```csv
time,ch1,ch2,ch3,...,ch16,command
```

## Dataset Descriptions

### raw/sample_eeg_data.csv
- **Source**: Simulated EEG data
- **Channels**: 1
- **Sampling Rate**: 256 Hz
- **Duration**: ~10 seconds
- **Commands**: NONE (0-2s), FOCUS (2-4s), RELAX (4-6s), mixed patterns
- **Use**: Primary dataset for development and integration testing

## Adding New Datasets

### From Public Sources
To add real EEG datasets (e.g., from PhysioNet):

1. Download dataset
2. Place in `raw/` directory
3. Convert to CSV format if needed (use scripts in `scripts/`)
4. Document dataset details below

### Creating Test Data
To create focused test data:

```bash
# Generate synthetic test signals
python scripts/generate_test_data.py --output data/test_cases/single_channel/
```

## Data Usage in Code

### Loading Data
```c
// Load from raw directory
FILE *fp = fopen("data/raw/sample_eeg_data.csv", "r");

// Load from test cases
FILE *fp = fopen("data/test_cases/single_channel/focus_only.csv", "r");
```

### Configuration
See `include/config.h` for data-related configuration:
- `SAMPLING_RATE`: 256 Hz
- `WINDOW_SIZE`: 256 samples (1 second)

## Real-World EEG Datasets (Optional Downloads)

### Recommended Public Datasets:

1. **PhysioNet EEG Motor Movement/Imagery Database**
   - URL: https://physionet.org/content/eegmmidb/1.0.0/
   - Format: EDF (convert to CSV)
   - Channels: 64
   - Tasks: Motor imagery, real movement

2. **BCI Competition Datasets**
   - URL: http://www.bbci.de/competition/
   - Various BCI paradigms
   - Already processed data available

## Notes

- Keep `raw/` directory read-only to preserve original data
- Test data should be small (<1MB) for fast CI/CD
- Validation data can be larger for comprehensive testing
- Generated synthetic data goes in `synthetic/`
